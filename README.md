# Civic Pulse - Municipal Meeting Summarizer

AI-powered summaries of local government meetings.

## Tech Stack

| Layer | Tech |
|-------|------|
| Frontend | TanStack Start |
| Backend | Convex |
| Auth | WorkOS AuthKit |
| UI | shadcn/ui + Tailwind v4 |
| AI | Claude API |
| Email | Resend |
| Deploy | Cloudflare Workers |
